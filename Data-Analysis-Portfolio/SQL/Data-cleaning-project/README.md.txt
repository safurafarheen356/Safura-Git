📘 Project Overview

This project focuses on cleaning raw layoff data using SQL to make it analysis-ready.
The dataset contains information about company layoffs, including details such as company name, industry, total employees laid off, and country.

The goal of this project is to:
Remove duplicate records
Standardize inconsistent text formats
Handle missing or null values
Convert and clean date formats
Prepare a final cleaned dataset for analysis

🛠️ Tools & Technologies Used

Database: MySQL
Language: SQL
